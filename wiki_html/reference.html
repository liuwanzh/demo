引用

初始化

引用又称为别名，可以用作对象的另一个名字。引用必须被初始化，单单int &refVal这样声明而不初始化是不行的。
注意引用与指针初始化的不同：
int val = 1024;
int *p = &val;
int &r = val;
一旦引用被定义，就不能再指向其他对象。上面例子中，若r=val2，并不会将r指向对象val2而非原先的val，而只是将val的值修改为val2.

指针和引用的主要区别

指针和引用的主要区别：
如果用一个引用给另一个引用赋值，那么改变的是被引用的对象而不是引用本身。比如:
int &r1 = val1; int &r2 = val2; r1 = r2时达到的效果是val1变成了val2的值，而r1依然是指向val1的。
但是指针则不同：int *p1 = &val1; int *p2 = &val2; p1=p2时达到的效果则是p1和p2都指向val2，而val1的值没有变化。
