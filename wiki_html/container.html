C++中的各种容器总结


map

map是关联性的容器，自动建立key对value的对应。使用map需要#include。

1.定义

map<keyType,valueType> mapname;例如map<int,string> student创建了一个student的map，其键值为int，value值为string。

2.插入

方法1：可以直接student[1]="liu"，这样的做法首先会检查student中是否有key为1的项，如果没有，则先新建一个key为1的项但value为缺省值，然后再为其赋值为“liu”。

方法2：student.insert(map < int,string >::valuetype(2,"Guo"));没有前面的效率问题。

任何类型（包括vector）都可以作为map的key和value，但是，作为key的类型必须重载了operator小于号，否则会报错。

3.查找是否含有key为特定值的项

map < int,string > iterator it = student.find(2); //查找key为2的项。
if(it==student.end()){ not found }
else {found,返回的iterator实际上是一个pair，it->first为key，it->second为value

4.删除

iterator erase(iterator it); //通过一个条目对象删除
iterator erase(iterator first, iterator last); //删除一个范围
size_type erase(const Key& key); //通过关键字删除
